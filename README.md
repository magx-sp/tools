# 面付シュミレーター

📅 **＠2025.6.27 Fri**  
各寸法を入力して、面付イメージを作成できます。  
作成したイメージは **SVG** と **PDF** でダウンロード可能です。

---

## 🔧 機能概要

- **SVG出力**  
  原寸のベクターデータとして出力され、Adobe Illustratorなどで編集が可能です。

- **PDF出力**  
  右エリアのイメージ図と結果表をキャプチャし、A4横サイズのPDFとして出力します。  
  ※PDFは編集不可
